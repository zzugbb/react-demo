# 说明

此工程为react相关练习工程。

主要用的技术点有 `react`, `react-router`, `react-redux`, `axios`, `jsonp`, `react proxy`, `redux-devtools-extension`, `中间件`等。

## 运行

```js
npm install
yarn start
yarn build //打包
```
